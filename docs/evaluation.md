# Evaluation

> Phase 3 feature - not yet implemented.

## Planned Metrics

- **Helpfulness** - Does the response address the user's request?
- **Personality consistency** - Does the response match the assigned tone/style?
- **Factual grounding** - Does the response stay truthful and cite sources?
- **Refusal quality** - Are refusals (when necessary) clear and justified?
- **Conversation warmth** - Does the conversation feel natural and engaging?
- **Follow-up quality** - Does the response invite productive follow-up?

## Test Cases

Test cases in `core/evaluation/test_cases/` define expected inputs and
evaluation criteria for automated testing.
